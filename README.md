# ***Sklep konsolowy w Javie***  
***Data wykonania: 06.02.2025***

## ***Opis projektu***  
Projekt przedstawia sklep konsolowy wykonany w Javie, korzystający z Maven jako systemu budowania.  
Celem projektu jest obsługa produktów i zamówień w trybie konsolowym, z możliwością zarządzania bazą danych SQLite.
